MediExpress

MedisExpress is myfirst project on which I am still working, this is a website or better a startup which is simple but makes ones order much trusted and easier.
MediExpress connects the customers to their local or global pharmaceuticles shops and an advanced delivery sytem helps in transporting the orders to the customers.
The main advantage of this startup is this delivery system is much faster and as local shops are also connected thus it helps the customer get their medicines from local and trustworthy medical shops.

Overview

This project was a deep dive into creating a robust, full-stack application. It involved designing a secure and scalable database, developing a highly performant backend, and building a fluid, responsive front-end user interface. The primary goal was to solve the problem of medical delivery from local and global shops to create a seamless user experience from start to finish.

Technologies Used

  • **Frontend**: HTML, JS, React, Next.js, Tailwind CSS, TypeScript
  • **Backend**: Node.js, Express, Firebase Functions
  • **Database**: Firestore
  • **Deployment**: Vercel, Firebase Hosting

My Role

I served as the lead developer on this project, responsible for the entire development lifecycle from conceptualization to deployment. This included writing all the frontend components, designing the database schema, setting up authentication, and deploying the application to a live server. I also managed version control using Git and collaborated with a designer to implement the UI/UX vision.

Outcome

The project is gradually becoming a complete success. My team has already made a few practical tests and is still working on a better and advanced features to be integrated in the app. This startup will cause a massive revolution in the delivery sectors.
